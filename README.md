# dont-go-for-broke
This is a repository for a money management app me and my brother are making for the 2025-2026 Lieutenant Governer's Computing Challenge #codingforgood

The app is designed for MacOS and iOS. The MacOS file is ```Don'tGoForBroke.zip``` in our latest release. If you build from source with Xcode you can get the iOS version. We couldn't publish the app to the App Store because we don't have an Apple Developer Program Membership.
